keta

Frontend Components for Kiwigrid Platform based AngularJS Apps

By developing and publishing keta we provide a set of components based on a flattened Bootstrap 3.x and running with AngularJS 1.2.x or AngularJS 1.3.x.

There will be visual components like buttons, form elements, navigation bars, but also AngularJS services for communicating with Kiwigrid backend services using a real-time web socket and some common features open-sourced as isolated modules.

We are highly interested in your feedback and are looking forward to seeing how keta will be used and enhanced in the future.
